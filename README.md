## :calling: Rodando o projeto

**Obs: ter o Yarn instalado na máquina**

Primeiro, clone o repositório e entre na pasta:
``` bash
git clone https://github.com/th1ag0-Zz/teste_tecnico_frontend.git
```
``` bash
cd teste_tecnico_frontend
```

Em seguida, instale as dependências com o comando ``` yarn ```.

Feito isso, dê o comando:
``` bash
  yarn start
```

Abrirá uma janela no seu navegador com o projeto rodando.
Caso não aconteça nada, basta acesar:
``` bash
http://localhost:3000
```
